## 🎨 Color Scheme

### Status Colors
- 🟢 **Success/Active** - `#50C878` - Owned, Active mortgage
- 🟠 **Warning** - `#FFB347` - In construction, Refinanced
- 🔴 **Error/Danger** - `#FF6B6B` - Defaulted, Expired
- 🔵 **Primary** - `#4A90E2` - In purchase, General actions
- ⚪ **Neutral** - `#9E9E9E` - Sold, Paid off

### Chart Colors
1. `#4A90E2` - Primary Blue
2. `#50C878` - Success Green
3. `#FFB347` - Warning Orange
4. `#FF6B6B` - Error Red
5. `#9B59B6` - Purple
6. `#3498DB` - Light Blue

## 📐 Spacing & Layout

### Card Spacing
- **Padding**: 24px (3 MUI spacing units)
- **Gap between cards**: 16px (2 MUI spacing units)
- **Card border radius**: 8px

### Grid Breakpoints
- **xs**: 0px - Mobile
- **sm**: 600px - Small tablet
- **md**: 900px - Tablet
- **lg**: 1200px - Desktop
- **xl**: 1536px - Large desktop

### Recommended Grid Layout
```tsx
// Properties list
<Grid container spacing={2}>
  <Grid item xs={12} sm={6} md={4} lg={3}>
    {/* 1 col mobile, 2 tablet, 3 desktop, 4 large */}
  </Grid>
</Grid>

// Details panels
<Grid container spacing={3}>
  <Grid item xs={12} md={6}>
    {/* 1 col mobile, 2 desktop */}
  </Grid>
</Grid>
```
