# ב. מעקב אחרי תשלומים

**עדיפות:** גבוהה  
**מורכבות:** נמוכה  
**מטרה:** לדעת מי שילם ומי לא שילם שכירות בכל חודש

---

## Backend

- [ ] **B-BE-1** — הוסף מודל `Payment` בסכמת Prisma  
  שדות: `id`, `leaseId`, `amount`, `dueDate`, `paidDate`, `paymentMethod`, `status` (PAID | PENDING | LATE), `notes`

- [ ] **B-BE-2** — כתוב migration ב-Prisma ליצירת טבלת `payments`

- [ ] **B-BE-3** — הוסף endpoint: `GET /payments?month=YYYY-MM`  
  מחזיר רשימת תשלומים צפויים לחודש נתון, כולל סטטוס תשלום

- [ ] **B-BE-4** — הוסף endpoint: `POST /payments/:id/mark-paid`  
  גוף הבקשה: `{ paidDate, paymentMethod, notes }`

- [ ] **B-BE-5** — הוסף endpoint: `GET /payments/debts`  
  מחזיר רשימת חובות פתוחים (תשלומים שלא שולמו עד היום)

- [ ] **B-BE-6** — הוסף endpoint: `GET /leases/:id/payment-history`  
  היסטוריית תשלומים לחוזה ספציפי

---

## Frontend

- [ ] **B-FE-1** — צור דף "מעקב תשלומים חודשי"  
  כולל בורר חודש (month picker) בראש הדף

- [ ] **B-FE-2** — הצג טבלה חודשית: דייר, נכס, סכום צפוי, סכום שולם, תאריך תשלום, סטטוס (Chip: שולם / ממתין / באיחור)

- [ ] **B-FE-3** — כפתור "סמן כשולם" על כל שורה — פותח Dialog עם שדות: תאריך תשלום, אמצעי תשלום, הערות

- [ ] **B-FE-4** — צור סקשן "חובות פתוחים" — רשימה של כל מי שחייב, עם סכום כולל

- [ ] **B-FE-5** — בדף הדייר — הוסף טאב "היסטוריית תשלומים" עם טבלה של כל התשלומים שלו

- [ ] **B-FE-6** — הוסף KPI בדשבורד הראשי: "תשלומים שהתקבלו החודש" / "ממתין לתשלום"
