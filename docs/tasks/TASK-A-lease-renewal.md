# א. מעקב חידוש חוזים

**עדיפות:** גבוהה  
**מורכבות:** בינונית  
**מטרה:** לקבל התראה מראש על חוזים שעומדים להסתיים

---

## Backend

- [ ] **A-BE-1** — הוסף endpoint: `GET /leases/expiring?months=X`  
  מחזיר רשימת חוזים שיסתיימו בתוך X חודשים מהיום  
  _שדות בתשובה: leaseId, tenantName, propertyAddress, endDate, monthlyRent_

- [ ] **A-BE-2** — הוסף שדה `renewalStatus` לטבלת `Lease` בסכמת Prisma  
  ערכים אפשריים: `PENDING` | `IN_PROGRESS` | `RENEWED` | `TERMINATED`

- [ ] **A-BE-3** — הוסף endpoint: `PATCH /leases/:id/renewal-status`  
  מאפשר עדכון סטטוס חידוש של חוזה

- [ ] **A-BE-4** — כתוב migration ב-Prisma לשדה החדש

---

## Frontend

- [ ] **A-FE-1** — צור דף / סקשן חדש: "חוזים שעומדים לפקוע"  
  ניווט: תפריט ראשי → "חוזים" → "עומדים לפקוע"

- [ ] **A-FE-2** — הוסף dropdown לסינון לפי טווח זמן  
  אפשרויות: "בחודש הקרוב" | "ב-3 חודשים" | "ב-6 חודשים"

- [ ] **A-FE-3** — הצג טבלה עם העמודות: שם דייר, כתובת, תאריך סיום, שכ"ד, סטטוס חידוש

- [ ] **A-FE-4** — הוסף אפשרות לשנות סטטוס חידוש ישירות מהטבלה (select / chip)

- [ ] **A-FE-5** — הוסף Badge בתפריט הראשי שמציג כמה חוזים עומדים לפקוע ב-60 יום הקרובים

---

- [ ] **A-QA-2** — E2E: ווידוא שהסינון לפי טווח זמן מחזיר את החוזים הנכונים
- [ ] **A-QA-3** — ווידוא שעדכון סטטוס חידוש נשמר ומתעדכן בטבלה
