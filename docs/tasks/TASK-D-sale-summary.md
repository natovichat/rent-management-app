# ד. סיכום עלויות בעת מכירת דירה

**עדיפות:** נמוכה  
**מורכבות:** גבוהה  
**מטרה:** להציג סיכום כולל של רווח / הפסד בעת מכירת נכס

---

## Backend

- [ ] **D-BE-1** — הוסף מודל `PropertyExpense` בסכמת Prisma  
  שדות: `id`, `propertyId`, `date`, `amount`, `description`, `category` (IMPROVEMENT | RECURRING | PURCHASE | SALE_COST), `notes`

- [ ] **D-BE-2** — כתוב migration ב-Prisma

- [ ] **D-BE-3** — הוסף CRUD endpoints לניהול הוצאות:
  - `GET /properties/:id/expenses?category=IMPROVEMENT`
  - `POST /properties/:id/expenses`
  - `PUT /expenses/:id`
  - `DELETE /expenses/:id`

- [ ] **D-BE-4** — הוסף endpoint: `GET /properties/:id/sale-summary`  
  מחזיר:
  - `purchasePrice` — מחיר רכישה
  - `salePrice` — מחיר מכירה
  - `totalImprovements` — סה"כ הוצאות השבחה
  - `totalRecurring` — סה"כ הוצאות שוטפות
  - `netProfit` — רווח נקי מחושב

- [ ] **D-BE-5** — הוסף שדות `purchasePrice` ו-`salePrice` למודל `Property` אם לא קיימים

- [ ] **D-FE-2** — כפתור "הוסף הוצאה" — Dialog עם שדות: תאריך, תיאור, קטגוריה (השבחה / שוטפת / אחר), סכום, הערות

- [ ] **D-FE-3** — צור סקשן/דף "סיכום מכירה" לנכס

- [ ] **D-FE-4** — הצג טבלת סיכום עם:

- [ ] **D-FE-5** — הוסף גרף עוגה / עמודות שמציג חלוקת ההוצאות לפי קטגוריה
